## Proyecto de Aula: Red social pascualina

### Propósito del Proyecto
Este proyecto surge de la necesidad de cerrar la brecha de comunicación informal en el entorno académico. Aunque existen plataformas institucionales, estas suelen ser rígidas y no fomentan la interacción orgánica entre todos.

El objetivo principal es **diseñar el modelo conceptual de una base de datos** robusta que soporte una red social estudiantil. Esta plataforma permitirá a la comunidad educativa:

* **Fomentar la Identidad:** Creación de perfiles detallados con intereses académicos, habilidades y hobbies.
* **Potenciar el Networking:** Conexión con compañeros para mentorías y seguimiento de intereses similares.
* **Intercambio de Conocimiento:** Un muro de actualizaciones para recursos técnicos, dudas de clase y noticias de los cursos.
* **Colaboración Grupal:** Espacios dedicados para grupos de estudio, equipos de hackatones y clubes de interés (programación, videojuegos, etc.).
* **Gestión de Eventos:** Coordinación de talleres, reuniones sociales y sesiones de estudio presenciales o virtuales.

Con este reto, buscamos aplicar conceptos avanzados de modelado de datos para resolver un problema real de interacción y apoyo mutuo dentro de la vida universitaria.
