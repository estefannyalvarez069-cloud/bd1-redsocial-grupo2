## Proyecto de Aula: Red social pascualina

### Propósito del Proyecto
Este proyecto surge de la necesidad de cerrar la brecha de comunicación informal en el entorno académico. Aunque existen plataformas institucionales, estas suelen ser rígidas y no fomentan la interacción orgánica entre todos.
Con este reto, buscamos aplicar conceptos avanzados de modelado de datos para resolver un problema real de interacción y apoyo mutuo dentro de la vida universitaria.
